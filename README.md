# ğŸ’¾ Start

### Clone project

```
git clone https://github.com/gacheon-sw-academy-2023/Timyloom-React.git
```
###  Installation
This package is distributed via npm.
```
$ npm install
```


### Usage

    npm run dev : Run project with vite 
    npm run build : Build a project
    npm run preview : Run after building a project
    npm run storybook : Run Storybook
    npm run build-storybook : Build a storybook
    npm run test : Run jest
    npm run coverage : Collect test coverage with Jest
